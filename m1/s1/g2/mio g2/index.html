Come creare un template per i post

di Mario Rossi - pubblicato il 25-10-2020

La prima cosa da fare è suddividere in blocchi logigi la pagina, identificando le parti comuni da applicare a tutte le pagine. Le varie aree verranno quindi suddivise in:

  • Contenuti di intestazione dei tag <header>
  • Contenuti centrali variabili del tag <main>
  • Contenuti di fine pagine comuni, da inserire nel tag <footer>

Formattazione del codice

Il testo dell'articolo dovrà essere formattato per identificare tutte le parti da selezionare con il CSS. Perciò i passi saranno:

  1. Progettare il markup assegnando ad ogni specifico testo il tag appropriato
  2. Scrivere l'HTML dei tag esterni a quelli interni dall'alto verso il basso
  3. Applicare stili CSS

L'applicazione degli stili CSS è cresciuta negli anni fino a raggiungere la diffusione pressochè totale.
